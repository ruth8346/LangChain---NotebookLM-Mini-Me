# 📚 NotebookLM Mini-Me – AI Research Assistant

פרויקט זה מציג סוכן בינה מלאכותית (AI Agent) מתקדם למחקר ואיסוף נתונים מהאינטרנט בזמן אמת.
הפרויקט מבוסס על **LangGraph** לניהול זרימת העבודה, **LangChain** לחיבור המודלים, ו-**Streamlit** כממשק משתמש.

---

## 🎯 מטרת הפרויקט
הסוכן מקבל נושא מחקר מהמשתמש, מבצע חיפוש מקורות וקישורים עדכניים ברשת, ומציג דוח מסכם ומיושר לימין (RTL).

### פיצ'רים מרכזיים:
* **Google Search:** חיפוש מידע חי מהרשת.
* **Human-in-the-Loop:** הסוכן עוצר וממתין לאישור אנושי לפני הפקת הדוח הסופי.
* **State Management:** שימוש בזיכרון של LangGraph לשימור מצב השיחה.

---

## 🚀 הוראות הרצה

### 1. דרישות קדם
ודאי שמותקנות הספריות הבאות:
pip install streamlit langchain-google-genai langgraph python-dotenv


### 2. הגדרת משתני סביבה

צרי קובץ בשם `.env` בתיקיית השורש והוסיפי את המפתח שלך:

GOOGLE_API_KEY="your_api_key_here"


### 3. הפעלת האפליקציה

הריצי בטרמינל:

streamlit run app.py
