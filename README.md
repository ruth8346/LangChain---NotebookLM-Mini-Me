# 📚 NotebookLM Mini-Me – עוזר מחקר אישי מבוסס AI

פרויקט זה מציג סוכן בינה מלאכותית (AI Agent) מתקדם למחקר ואיסוף נתונים מהאינטרנט בזמן אמת. 
הפרויקט מבוסס על **LangGraph** לניהול זרימת העבודה, **LangChain** לחיבור המודלים, ו-**Streamlit** כממשק משתמש גרפי.

---

## 🎯 מטרת הפרויקט
הסוכן מקבל נושא מחקר מהמשתמש, מבצע חיפוש מקורות וקישורים עדכניים ברשת, ומציג דוח מסכם ומפורט.

### פיצ'רים מרכזיים:
* **Google Search:** חיפוש מידע חי ועדכני מהאינטרנט.
* **Human-in-the-Loop:** הסוכן עוצר את הריצה וממתין לאישור אנושי לפני הפקת הסיכום הסופי, לביצוע בקרת איכות.
* **State Management:** שימוש בזיכרון של LangGraph לניהול מצב השיחה בצורה רציפה.

---

## 🚀 הוראות הרצה

### 1. דרישות קדם
ודאי שמותקנות הספריות הבאות בסביבה הוירטואלית שלך:
pip install streamlit langchain-google-genai langgraph python-dotenv


### 2. הגדרת משתני סביבה

צרי קובץ בשם `.env` בתיקיית השורש והוסיפי את המפתח שלך:

GOOGLE_API_KEY="your_api_key_here"


### 3. הפעלת האפליקציה

הריצי בטרמינל את הפקודה הבאה:

streamlit run app.py
