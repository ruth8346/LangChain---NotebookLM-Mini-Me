# 📚 NotebookLM Mini-Me – AI Research Assistant

פרויקט זה מציג סוכן בינה מלאכותית (AI Agent) מתקדם למחקר ואיסוף נתונים מהאינטרנט בזמן אמת, בהשראת הקונספט של NotebookLM. 
הפרויקט מבוסס על **LangGraph** לניהול זרימת העבודה (Graph State), **LangChain** לחיבור המודלים, ו-**Streamlit** כממשק משתמש גרפי.

---

## 🎯 מטרת הפרויקט וארכיטקטורה

מטרת הסוכן היא לקבל נושא מחקר מהמשתמש, לבצע חיבור לאינטרנט לצורך שליפת מקורות וקישורים עדכניים, ולייצר דוח מסכם ומיושר לימין (RTL).

### פיצ'רים מרכזיים:
* **Google Search Tool Integration:** שימוש בכלי חיפוש מובנה של Gemini לשליפת מידע חי מהרשת (כולל תמיכה מלאה בסינון נטפרי).
* **Human-in-the-Loop (HITL):** הגרף בנוי עם נקודת עצירה מובנית (`interrupt_after`). הסוכן אוסף את המידע, עוצר, וממתין לאישור אנושי אקטיבי בממשק לפני הפקת הסיכום הסופי.
* **State Management:** שימוש ב-`MemorySaver` של LangGraph לשימור מצב השיחה והזיכרון של הסוכן לאורך שלבי הריצה השונים ב-Streamlit.

---

## 🚀 איך להריץ את הפרויקט?

### 1. התקנת סביבה ודרישות קדם
יש לוודא שמותקנת סביבה וירטואלית וכל הספריות הנדרשות. להפעלה מהטרמינל:
```bash
pip install streamlit langchain-google-genai langgraph python-dotenv
2. הגדרת משתני סביבה
יש ליצור קובץ בשם .env בתיקיית השורש של הפרויקט ולהוסיף את מפתח ה-API שלך:

קטע קוד
GOOGLE_API_KEY="your_gemini_api_key_here"
3. הרצת הממשק הגרפי
כדי להפעיל את האפליקציה, מריצים את הפקודה הבאה בטרמינל:

Bash
streamlit run app.py

🛠️ מבנה הקבצים בפרויקט
agent_1.py – הגדרת הגרף, כלי החיפוש (Tools), ה-Prompt המערכתי של הסוכן ונקודות העצירה האנושיות.

app.py – ממשק המשתמש הגרפי שנבנה ב-Streamlit, המנהל את ה-Session State ומציג את הפלט בעיצוב עברית תקין (RTL).

.env – קובץ הגדרות סודיות למפתח ה-API (לא להעלאה לגיט).
